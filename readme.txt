== Description ==
Features:
 * Simple Domain Search Form easy to Use.
 * Place [Domain Search], [Domain Transfer] or [Vanity Domains] on any Page or Post.
 
 * You can add custom heading or call out to the search forms via the Hyper Domains page under settings.
 
 * If you have an Hyper.Domains re-seller account or sub re-seller account you can add your Reseller domain.
 
 * You can sign-up for a reseller account here:  <a href="http://resellers.hyper.domains/reseller.php?action=show_signupform">Create a Reseller Account!</a>
 

== Installation ==

1. Extract `hyper-domains-search.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [Domain Search], [Domain Transfer] or [Vanity Domains] on any Page or Post
4. Can add custom heading or call out to the search forms via the Hyper Domains page under settings.
5. If you have an Hyper.Domains re-seller account or sub re-seller account you can add your Reseller domain. 


== Frequently Asked Questions ==

= How to use with short-code ? =
Place [Domain Search], [Vanity Domains] or [Domain Transfer] on any Page or Post

= Can I get my own re-seller website ? =
Yes. You can sign up for your very own re-seller website and insert your own URL for the search results.

= Can I have my own re-sellers ? =
Yes. You may have your own re-sellers and your re-sellers can too and so on and so on.

= Can I customize the look of the re-seller website ? =
Yes. You can edit all CSS, Content and Markup on the Re-seller Website.
= Can I customize the look of the wp-plugin search form ? =Yes. To make modifications to the form, go to css/style.css and customize the form. The form will pick up default wordpress theme form styles as well.
= Can I direct the wp search form to another registrar or affiliate link  ? =
Yes. But you would need to modify the code and are free to do so.
